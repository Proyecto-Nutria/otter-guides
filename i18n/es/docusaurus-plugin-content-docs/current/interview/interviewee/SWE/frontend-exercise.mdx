En este tipo de ejercicio, generalmente se te presentará algún componente de UI/UX que deberás
construir ya sea utilizando CSS+HTML+JS básicos o mediante algún framework como React, Angular, Vue,
Svelte, ... Por lo general, tendrás entre 30 minutos y 1 hora para completar este ejercicio.

### Ejemplo de ejercicio

#### Enunciado
1. Obtén datos de https://swapi.dev/api/people/
2. Utilizando los datos de nuestra API, imprime una lista de personajes de Star Wars.
3. Muestra información sobre cada personaje al hacer clic en su nombre en la lista.
4. [Solución -> stoic-neumann-f789mi - CodeSandbox](https://codesandbox.io/s/stoic-neumann-f789mi?file=/src/App.js)
