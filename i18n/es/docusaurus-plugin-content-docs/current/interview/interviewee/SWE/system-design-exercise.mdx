title: Ejercicio de diseño de sistemas
---

En este tipo de ejercicio, se espera que generes un diseño de alto nivel para un sistema, mostrando sus diferentes componentes e infraestructura, considerando ambos lados (front-end y back-end). Usualmente, tendrás entre 30 minutos y 1 hora para completar este ejercicio. Aunque proporcionar un diseño de alto nivel es el enfoque principal de este ejercicio, también puede que te pidan que expliques algunas partes en detalle. Por ejemplo, ¿cómo crearás la API REST? ¿Qué tecnología preferirás para algunas tareas y por qué?
